\# Ứng dụng Học tiếng Anh



Ứng dụng Spring Boot dành cho việc học tiếng Anh với các tính năng thu thập từ vựng.



\## Cấu trúc dự án



\- `crawler/` - Mô-đun thu thập dữ liệu từ vựng tiếng Anh

\- `application/` - Mô-đun ứng dụng Spring Boot chính



\## Công nghệ Stack



\- Java 17

\- Spring Boot

\- Maven (Nhiều mô-đun)
